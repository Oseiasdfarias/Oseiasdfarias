## About Me

I am a Full-Stack Backend Developer and Electrical Engineer based in São Paulo, Brazil, with a Master's in Electrical Engineering from UFABC (expected 2027). Specializing in crafting robust, scalable REST APIs using Java, Python, Spring Boot, and FastAPI, I leverage my expertise in Spring Security, JPA, MySQL, and PostgreSQL to build secure, high-performance systems. My background includes systems integration through the ONE - Oracle Next Education program, where I honed practical skills in API development, and I contributed to the AlumniOne Committee by sharing tech insights via community newsletters. Passionate about transforming complex data into intelligent, business-driving solutions, I blend engineering principles with AI/ML to automate workflows and optimize operations. As a former UHT Maintenance Intern at Grupo Piracanjuba, I applied Python and Pandas for production line analysis, reducing downtime and boosting efficiency. Currently, I'm advancing research in Artificial Antifragility at FUNDEP/UFABC, integrating FPAA, AI, and HIL simulation for resilient control systems.

- Currently working on: Serverless SDR agents with OpenAI, FastAPI, and Vercel Deploy
- Learning: Kubernetes and Deep Learning and LLMs
- Open to: Backend APIs, data engineering, or open-source AI collaborations
- Ask me about: Artificial Antifragility, FastAPI, or Python & Java
- Reach me: oseias.dfarias@gmail.com
