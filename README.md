## About Me

Full-Stack Backend Developer and Electrical Engineer based in São Paulo, pursuing a Master's in Electrical Engineering at UFABC (expected 2027). Specialized in building scalable REST APIs with Java, Python, Spring Boot, and FastAPI. Expertise in Spring Security, JPA, MySQL, and PostgreSQL. Experience in systems integration through Oracle's ONE program and AI/ML applications for workflow automation. Former intern at Grupo Piracanjuba applying Python for production optimization. Currently researching Artificial Antifragility, integrating FPAA, AI, and HIL simulation for resilient control systems.

- Learning: Kubernetes and Deep Learning and LLMs
- Open to: Backend APIs, data engineering, or open-source AI collaborations
- Ask me about: Artificial Antifragility, FastAPI, or Python & Java
